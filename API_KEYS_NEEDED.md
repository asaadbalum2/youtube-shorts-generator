1. Go to: `https://www.pexels.com/api/`

1. Go to: `https://pixabay.com/api/docs/`
2. Sign up (free)
3. Copy your API key
4. In Replit Secrets: `PIXABAY_API_KEY` = your key

**Note:** You need AT LEAST ONE of these, preferably both for better variety.
